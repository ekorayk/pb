## Teknik Bilgi Tabanı

### Parlak Beton Uygulaması — Teknik Standartlar

**Parlaklık Seviyeleri (Mohs/Gloss):**

| Seviye | Gloss | Kullanım |
|--------|-------|---------|
| Mat | 0-20 | Endüstriyel, yüksek trafik |
| Yarı Parlak | 20-50 | Depo, imalathane |
| Parlak | 50-80 | Showroom, ofis |
| Ayna | 80+ | Premium perakende, hastane |

**DCOF (Dynamic Coefficient of Friction) Standartları:**

- ANSI B101.1: Islak zemin ≥ 0.42 DCOF
- NFSI: Yüksek sürtünme ≥ 0.65 DCOF
- Türkiye TS 202: ≥ 0.40 (ıslak)

**Lityum Silikat Kimyası:**

- Beton gözeneklerine nüfuz: 3-6mm derinlik
- Kimyasal reaksiyon: SiO2 + Ca(OH)2 → C-S-H jeli
- Yoğunlaştırıcı etki: %40 aşınma direnci artışı
- Uygulama sıcaklığı: +5°C ile +35°C arası

### Uygulama Protokolü

```
1. Zemin Hazırlığı
   → Mevcut kaplama analizi (sertlik testi)
   → Kirletici temizliği (yağ, boya, yapışkan)
   → Beton sertlik ölçümü (minimum 3000 PSI ideal)

2. Kaba Öğütme (Metal Bond)
   → 16/30/60 grit sırası
   → Su bazlı soğutma sistemi
   → Çatlak ve çukur onarımı

3. İnce Öğütme (Resin Bond)
   → 100/200/400 grit
   → Lityum silikat penetrant uygulama
   → İşlem süresi: 4-8 saat bekleme

4. Parlatma
   → 800/1500/3000 grit
   → Son yüzey koruyucu (guard/densifier)

5. Kalite Kontrolü
   → Gloss ölçümü (glossmetre)
   → DCOF testi
   → Fotoğraflı belgeleme
```
