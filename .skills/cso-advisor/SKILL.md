## Parlak Beton Satış Çerçevesi

### İdeal Müşteri Profili (ICP)

**Birincil Hedef:**

- Fabrika / üretim tesisi yöneticisi
- Depo / lojistik merkezi sorumlusu
- Alışveriş merkezi / hastane teknik direktörü
- Ofis binası yöneticisi

**Nitelik kriterleri:**

- Alan: minimum 500m²
- Bütçe: 150K ₺+
- Karar verici: teknik ya da satın alma direktörü

### Keşif Öncesi Lead Nitelendirme

```
Sorulacaklar (form veya ilk telefonda):
  □ Zemin alanı (m²)?
  □ Mevcut zemin durumu?
  □ İstenen tamamlanma tarihi?
  □ Projeyi kim onaylıyor (bütçe sahibi)?
  □ Benzer proje teklifi aldı mı?
```

**Kırmızı bayraklar (keşife gitme):**

- 200m² altı alan
- "Araştırıyoruz" + aciliyet yok + bütçe belirsiz
- Sadece fiyat isteyen (keşif talebi yok)

### Satış Süreci

```
1. Lead Giriş (0-2 saat)
   → Form / WhatsApp / telefon
   → Nitelendirme soruları
   → Keşif randevusu → takvim

2. Keşif (saha ziyareti)
   → Zemin analizi ve fotoğraflama
   → Müşteri beklentisi netleştirme
   → Keşif tutanağı imzalama

3. Teklif (48 saat içinde)
   → AI destekli teklif hazırlama
   → PDF + Canva kapak
   → E-posta + WhatsApp gönderim

4. Takip (3-7-14 gün)
   → Güvenli baskı: "Sorunuz var mı?"
   → İtiraz varsa çözüm sun
   → Karar bekleme süresi anlaşmak

5. Kapama
   → Sözleşme hazırlama (CLO onaylı şablon)
   → Avans + uygulama başlangıç tarihi
```

### Fiyatlama Çerçevesi (CFO onaylı)

```
Temel hesap:
  Malzeme maliyeti × 3 = minimum fiyat
  Pazar fiyatı araştırması × 0.9 = rekabetçi üst sınır

Alan bazlı iskonto (COO onaylı):
  500-1000 m²:   standart fiyat
  1000-3000 m²:  %5-8 iskonto
  3000+ m²:      müzakereye açık (%10-15)
```
