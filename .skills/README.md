# Skills

Bu klasör, proje genelinde yeniden kullanılabilir beceri (skill) modüllerini içerir.

## Skill Nedir?

Her skill, belirli bir görevi nasıl yapacağını açıklayan bir `SKILL.md` dosyası ve isteğe bağlı yardımcı dosyalar içerir.

## Mevcut Skills

| Klasör | Açıklama |
|--------|----------|
| `seo-audit/` | Sayfa SEO denetimi |
| `image-optimize/` | Görsel optimizasyon rehberi |
| `component-builder/` | Bileşen oluşturma şablonu |

## Yeni Skill Eklemek

```
.skills/
└── skill-adi/
    ├── SKILL.md          # Zorunlu — talimatlar ve açıklama
    ├── scripts/          # Opsiyonel — yardımcı scriptler
    ├── examples/         # Opsiyonel — örnek uygulamalar
    └── resources/        # Opsiyonel — şablonlar, varlıklar
```
