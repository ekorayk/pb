```

AŞAMA 1: KEŞİF (Saha Ziyareti)

```

## Aşama 4: Kalite Kontrol

```
KALİTE KONTROL TUTANAĞI

□ Gloss ölçümü yapıldı (glossmetre)
    Hedef: [____] | Ölçülen: ____
    Tutarsız bölge var mı: □ Yok □ Var → [not]

□ DCOF testi yapıldı (ıslak zemin)
    Standart: ≥ 0.42 | Ölçülen: ____
    PASS □  FAIL □

□ Görsel inceleme
    □ Eşit parlaklık dağılımı
    □ Çizik/leke yok
    □ Kenar temizliği

□ Müşteri memnuniyeti görüşmesi yapıldı
□ Kabul fotoğrafları çekildi (minimum 10)
□ Müşteri imzası alındı: □ Evet
```
