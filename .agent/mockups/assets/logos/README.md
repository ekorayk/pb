# Logo Varlıkları

Bu klasöre Parlak Beton logo dosyalarını ekleyin.

## Beklenen Dosya İsimleri

Aşağıdaki isimleri kullanırsanız mockup otomatik olarak görüntüler:

| Dosya | Kullanım |
|-------|---------|
| `logo.svg` | Navbar ana logo (SVG önerilir) |
| `logo.png` | Navbar ana logo (PNG alternatif) |
| `logo-white.svg` | Beyaz versiyon (koyu arka plan için) |
| `logo-white.png` | Beyaz PNG alternatif |
| `logo-dark.svg` | Siyah versiyon (açık arka plan için) |
| `favicon.svg` | Browser favicon |
| `favicon.png` | Browser favicon (PNG) |

## Desteklenen Formatlar

- **SVG** → En iyi kalite, sonsuz ölçekleme
- **PNG** → Şeffaf arka planlı yüksek çözünürlük (2x/3x önerilir)
- **WebP** → Küçük dosya boyutu

## Mockup Entegrasyonu

Logo dosyalarını bu klasöre ekledikten sonra mockup otomatik olarak:

- Navbar'da logo görüntüler (PB yazısı yerine)
- Footer'da küçük logo gösterir

## Tavsiye Boyutlar

- Navbar logo: 160×40px (SVG veya 2x PNG: 320×80px)
- Favicon: 32×32px SVG veya 512×512px PNG
