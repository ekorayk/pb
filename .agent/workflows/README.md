# Workflows

Bu klasör, proje geliştirme süreçleri için otomasyon iş akışlarını içerir.

## Kullanım

Her `.md` dosyası bir iş akışını tanımlar. Slash komutu olarak çağrılabilir:

```
/workflow-adi
```

## Mevcut Workflows

| Dosya | Açıklama |
|-------|----------|
| `dev-server.md` | Geliştirme sunucusunu başlatır |
| `build.md` | Prodüksiyon build alır |
| `deploy.md` | Projeyi deploy eder |
