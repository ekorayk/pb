# parlakbeton.com

Parlak Beton kurumsal web sitesi.

## Hızlı Başlangıç

```bash
npm install
npm run dev
```

## Klasör Yapısı

```
parlakbeton.com/
├── .agent/          # AI ajan iş akışları ve bağlam
├── .skills/         # Yeniden kullanılabilir beceri modülleri
├── public/          # Statik dosyalar
├── src/             # Kaynak kodlar
├── CLAUDE.md        # AI asistan talimatları
└── README.md        # Bu dosya
```

## Geliştirici Komutları

| Komut | Açıklama |
|-------|----------|
| `npm run dev` | Geliştirme sunucusu |
| `npm run build` | Prodüksiyon build |
| `npm run deploy` | Deploy |
